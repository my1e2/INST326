# INST326 — Object-Oriented Programming in Python

Repository containing coursework, labs, projects, lecture materials, and programming exercises completed throughout INST326.

## Overview

This course focused on object-oriented software development using Python. Topics included class design, encapsulation, inheritance, polymorphism, collaborative development practices, file handling, and database interaction.
